Cara Jalankan script API Flask dan API Motoko.


Cara jalanakan script API Flask:

1. Download file rar API-Flask-KuSehat
2. Extrak file kedalam folder baru yaitu KuSehat-API
3. Masuk kedalam folder KuSehat-API,kemudian pip install -r requirements.txt
4. Jalankan web server dengan cara python app.py.

Endpoint untuk Luno: http://localhost:8000/luno
Endpont untuk API KuSehat: http://localhost:8000/

Cara jalanlan script API Motoko:

1. Copy Paste folder KuSehat-Web3-Motoko kedalam folder WSL Ubuntu home/namakamu contoh home/rauf
2. Sebelum jalankan Website KuSehat di Motoko,harus install Motoko dan WSL Ubuntu terlebih dahulu: https://internetcomputer.org/docs/motoko/install.

3. Jalankan website didalam folder KuSehat-Web3-Motoko, dengan ketik di Ubuntu: dfx start --clean --background kemudian ketik dfx deploy.
